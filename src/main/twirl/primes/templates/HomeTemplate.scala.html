@home.context.title

The @home.context.appcode application is a primes number service with API and arts features. A large set of primes number is already computed and ready for any query.

  • @home.randomPrimeNumber is a randomly chosen prime number
  • @home.computedCount already computed primes number in store
  • @home.maxKnown is the highest computed prime number

Take a look to the Swagger API specification to use this service or go to the @home.context.appcode github repository if you want to contribute.
